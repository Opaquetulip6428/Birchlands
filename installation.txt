To install, make sure your Minecraft: Java Edition version is 1.17 or 1.17.1.

- Download this as a .zip folder, then unzip the folder.

- When the folder is unzipped, take out the folder within it called "The Birchlands"

- Create a new world, click "Data Packs"

- Drag the specified folder from earlier into the designated area.

- Load the world.

To install on a pre-existing world, simple download the .zip, untip the folder, then place the "The Birchlands" folder in the datapacks folder. All new chunks will have the new worldgen.
